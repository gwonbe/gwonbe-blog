Location 객체

  • Location 객체는 객체가 연결된 장소(URL)를 표현한다.
  • Location 객체에 변경을 가하면 연결된 객체에도 반영된다.
  • Document와 Window가 이런 Location을 가지고 있으며 각각 Document.location, Window.location으로 접근할 수 있다..

Location 인터페이스의 다양한 속성과 예제

속성 설명
location.href 온전한 URL을 값으로 하는 DOMString이다. 해당 주소로 이동한다.
location.protocol URL의 프로토콜을 가져오는 DOMString으로, 마지막의 ':'도 포함한다.
location.host URL의 호스트를 가져오는 DOMString으로 호스트명, ':', 포트번호를 포함한다.
location.hostname URL의 도메인을 가져오는 DOMString이다.
location.port URL의 포트 번호를 가져오는 DOMString이다.
location.pathname '/' 문자 뒤 URL의 경로를 값으로 하는 DOMString이다.
location.search '?' 문자 뒤 URL의 쿼리스트링을 값으로 하는 DOMString이다. 브라우저에서 URLSearchParams.get()과 URL.searchParams를 사용해서 인자를 추출할 수 있다.
location.hash '#' 문자 뒤 URL의 프래그먼트 식별자를 값으로 하는 DOMString이다.
location.username 도메인 이름 이전에 명시된 사용자명을 값으로 하는 DOMString이다.
location.password 도메인 이름 이전에 명시된 비밀번호를 값으로 하는 DOMString이다.
location.origin 읽기 전용으로, 지정한 장소 오리진의 표준 형태를 값으로 하는 DOMString이다.

소수점 다루기

방법 설명
Math.floor(myNumber) 변수 myNumber를 소수점 내림하여 정수로 반환한다.
Math.ceil(myNumber) 변수 myNumber를 소수점 올림하여 정수로 반환한다.
Math.round(myNumber) 변수 myNumber를 소수점 반올림하여 정수로 반환한다.
myNumber.toFixed() 변수 myNumber의 소수점 전체를 반올림한다.
myNumber.toFixed(x) 변수 myNumber의 소수점을 x로 고정한다. 만약 x가 현재 변수의 소수점 자리보다 적다면 x 자리까지 반올림되고, 소수점 자리보다 많다면 x 자리까지 0으로 채워진다.
