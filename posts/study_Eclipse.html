Eclipse 정리

Eclipse에서 Apache Tomcat 연동하기

  1. 아파치 톰캣 설치하기

    찾기 쉽도록 로컬디스크C 하위에 저장하기

  2. 아파치 톰캣 경로 지정하기

    이클립스에서 window > Preferences > Server > Runtime Environment 들어가기
    Add 클릭해서 Apache 폴더에서 버전 선택 후 Next 누르기
    Browse 클릭해서 C:\apache-tomcat-9.0.70 경로 설정한 후 Finish 누르기

  3. 아파치 톰캣 선택하기

    프로젝트 우클릭 후 Properties 들어가기
    Java Build Path > Libraries > Add Library 들어가기
    Server Runtime 선택 후 나오는 아파치 톰캣 선택하기

UTF-8로 인코딩 설정하기

  1. window > Preferences > Web 들어가기
  2. CSS Files, HTML Files, JSP Files 인코딩 UTF-8로 바꾸기
  3. XML > XML Files 인코딩 UTF-8로 바꾸기
