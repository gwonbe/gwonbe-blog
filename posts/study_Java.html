JAVA 정리

[ JDK & JRE ] 비교하기

  • JDK
    • Java Development Kit의 약자로 자바 개발 키트를 말한다.
    • 자바 프로그램을 개발하는데 필요한 개발 도구들을 포함한다.
    • jre, javac, javadoc, jar 등이 포함된다.
  • JRE
    • Java Runtime Environment의 약자로 자바 실행 환경을 말한다.
    • JRE로는 개발이 불가능하며 오직 실행만 가능하다.
    • 자바로 개발한 프로그램을 실행하는데 필요한 자바가상머신, 라이브러리, API를 포함한다.

현재 날짜 구하기

코드

import java.util.Date; import java.time.LocalDate; import java.time.format.DateTimeFormatter; public class Test { public static void main(String[] args) { // import nothing! System.out.println("\n[01] ------------------------"); System.out.println(new java.util.Date()); // import java.util.Date; System.out.println("\n[02] ------------------------"); System.out.println(new Date().toLocaleString()); // import java.time.LocalDate; System.out.println("\n[03] ------------------------"); System.out.println(LocalDate.now()); // import java.time.LocalDate; import java.time.format.DateTimeFormatter; System.out.println("\n[04] ------------------------"); LocalDate now = LocalDate.now(); DateTimeFormatter formatter = DateTimeFormatter.ofPattern("YYYY/MM/dd"); String formatedNow = now.format(formatter); System.out.println(formatedNow); } }

결과
