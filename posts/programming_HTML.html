간단한 설명

  • innerText : 해당 요소를 제외한 자손의 렌더링된 텍스트 콘텐츠를 가져온다.
  • outerText : 해당 요소에 자손 태그 요소를 포함하여 텍스트 콘텐츠를 가져온다.
  • innerHTML : 해당 요소의 태그를 제외한 자손 태그의 값만 가져온다.
  • outerHTML : 해당 요소의 태그 값과 속성 값을 함께 가져온다.
