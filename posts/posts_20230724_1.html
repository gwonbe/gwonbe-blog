JSP 스크립트 태그에 대하여

스크립트 태그란?

  • JSP 페이지가 서블릿 프로그램에서 서블릿 클래스로 변환할 때 적용한다.
  • JSP 컨테이너가 자바 코드가 삽입되어 있는 스크립트 태그를 처리하고 나머지는 HTML 코드나 일반 텍스트로 간주한다.

스크립트 태그의 종류

종류 태그 형식 사용 용도
지시어 (directives) 
<%@    %>
 JSP 페이지의 속성을 지정
선언 (declaration) 
<%!    %>
 소속 변수 선언과 메소드 정의
표현식 (expression) 
<%=    %>
 변수, 계산식, 함수 호출 결과를 문자열 형태로 출력
스크립트릿 (scriptlet) 
<%     %>
 JAVA 코드를 기술
주석 (comments) 
<%--   --%>
 JSP 페이지의 설명 추가
