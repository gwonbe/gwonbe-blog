python & pip 설치 확인 방법

  1. 파이썬 설치 확인하기
    1. 명령프롬프트에 'where python' 입력해서 설치 경로 확인하기
    2. 설치 경로를 복사하기
    3. 파일 탐색기에 들어가서 경로 중 '\python.exe'는 지우고 붙여넣기
    4. 경로 이동 후 Scripts 폴더 열어서 pip 있는지 확인하기
  2. 환경 변수 설정하기
    1. 윈도우 검색창이 '고급 시스템 설정 보기' 들어가기
    2. 고급에 환경변수 클릭하기
    3. 아래 시스템변수에 'Path' 선택 후 편집 들어가기
    4. 새로만들기 클릭 후 아래 2개 추가하기
      • python 설치 경로
      • python 설치 경로\Scripts

이제 명령크롬프트에서 python, pip, pip install pandas 등 명령이 잘 수행된다.
이 과정이 잘 안 됐다면 파이썬 다시 설치해야한다.
재설치 시 pip 체크를 꼭 확인하자.
