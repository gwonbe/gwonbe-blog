수평 수직 정렬

화면의 정가운데에 콘텐츠를 표시해보자.

위와 같이 화면 크기가 변해도 콘텐츠를 수평 수직 정렬해주는 코드는 다음과 같다.

HTML CSS
<div id="container"> <h1>Hello</h1> <h1>World</h1> </div> #container{ width: 300px; background-color: lavender; text-align: center; /* 수평 수직 정렬 */ position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }
